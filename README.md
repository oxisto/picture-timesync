# Requirements

## Linux

* sudo add-apt-repository ppa:pyexiv2-developers/ppa
* sudo apt-get update
* sudo apt-get install python-pyexiv2

## OS X

* brew install exiftool
* brew install pygobject3
* brew install --with-introspection gexiv2
* brew install pyexiv2
